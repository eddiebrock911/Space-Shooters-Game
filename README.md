## рҹҢҢ About

**Space Shooters Game** is a fast-paced browser game built using **HTML, CSS, and JavaScript**.

Take control of your spaceship, destroy incoming enemies, avoid collisions, and survive for the highest possible score.

This project demonstrates core JavaScript game development concepts including collision detection, animations, keyboard controls, score tracking, and game logic.
